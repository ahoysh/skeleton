Skeleton
========

~~~
   skeleton 1.0.0 - Ahoy Matey!! Start projects faster using skeleton templates

   USAGE

     skeleton <command> [options]

   COMMANDS

     search <name>       Searches for a skeleton template   
     start [name]        Starts an empty skeleton template  
     register            Registers a skeleton template      
     help <command>      Display help for a specific command

   GLOBAL OPTIONS

     -h, --help         Display help                                      
     -V, --version      Display version                                   
     --no-color         Disable colors                                    
     --quiet            Quiet mode - only displays warn and error messages
     -v, --verbose      Verbose mode - will also output debug messages
~~~

    Installation:

~~~
        npm install -g
~~~
